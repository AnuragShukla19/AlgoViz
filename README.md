> **AlgoViz** — *See algorithms in motion.* An interactive visualizer for **data structures and algorithms** with step-by-step animations. Built for students and developers who want to learn DSA visually.

## How to contribute

> I built this project with Vanilla JS to keep it simple and easy to contribute to. You can help in many ways:

### You can contribute to this project in many ways
 1. Create an issue if you find any bug.
 2. Work on an existing issue and send a PR.
 3. Add another data structure or algorithm and send a PR.
 4. Create an issue for grammatical or spelling mistakes.

> *I'd be glad to see a notification saying someone created a Pull Request. I promise to review it.*
